# vue3template npm版本（18.19.0）

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 打包web版
```
npm run build
```

### 打包electron版客户端（直接生成文件夹）
```
npm run build:electron
```

### 打包electron版客户端（生成安装包）
```
npm run electron:build -- --win --x64 --ia32
```
